# Scholar Companion

Public distribution channel for the **Scholar Companion** Zotero plugin (built artifact + auto-update manifest).

- **Install / download:** grab `scholar-companion.xpi` from [the latest release](https://github.com/dsmoz/scholar-companion/releases/latest), then in Zotero: **Tools → Add-ons → gear → Install Add-on From File**.
- **Auto-update:** Zotero reads [`update.json`](update.json) (the plugin's `update_url`) and offers new versions automatically.

Source code lives in the private monorepo. This repo holds only the built `.xpi` and the update manifest — no source.
