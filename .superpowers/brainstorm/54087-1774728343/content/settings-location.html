Where should the similarity filter live?

User sets minimum label threshold — only documents at or above it appear in Related Documents.

A

Plugin Preferences (existing settings panel)

Added alongside existing settings like "max related docs". Set once, applies globally. User opens Zotero → Tools → zotero-plugin prefs.

B

Inline filter in the Related Documents panel

Small dropdown or segmented control directly above the list. Instant feedback — change the filter and the list updates immediately. No need to open settings.

C

Both — inline for quick changes, prefs for default

Preferences stores the default. The inline control overrides it for the current session. Slightly more complexity but maximum flexibility.
